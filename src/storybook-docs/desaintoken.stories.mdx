# Desain Token
Desain token **Jabar Digital Service Sistem Desain** merupakan sebuah abstraksi dari properti visual seperti warna, *background* dan *border* yang di aplikasikan dengan cara menyimpan token sebagai *variabel*. Desain token diperuntukkan untuk menciptakan konsistensi *User Interface* dalam berbagai *platform*.

## Warna

### Teks
Gunakan format `text-${colorName}-${variant}`.
```html
<p class="text-green-500">
  Lorem ipsum dolor sit amet.
</p>
<p class="text-blue-gray-700">
  Lorem ipsum dolor sit amet.
</p>
```
<Canvas withSource={ SourceState.NONE }>
  <p className="text-green-500 font-sans-1">
    Lorem ipsum dolor sit amet.
  </p>
  <p className="text-blue-gray-700 font-sans-1">
    Lorem ipsum dolor sit amet.
  </p>
</Canvas>

### Background
Gunakan format `bg-${colorName}-${variant}`.
```html
<div class="bg-blue-300 text-white">
  Lorem ipsum dolor sit amet.
</div>
<div class="bg-yellow-800 text-white">
  Lorem ipsum dolor sit amet.
</div>
```
<Canvas withSource={ SourceState.NONE }>
  <div
    className="bg-blue-300 text-white font-sans-1"
    style={{ 'padding': '2rem'}}>
    Lorem ipsum dolor sit amet.
  </div>
  <div
    className="bg-yellow-800 text-white font-sans-1"
    style={{ 'padding': '2rem'}}>
    Lorem ipsum dolor sit amet.
  </div>
</Canvas>

### Border
Gunakan format `border-${colorName}-${variant}`.
```html
<div class="border-black">
  Lorem ipsum dolor sit amet.
</div>
<div class="border-pink-300">
  Lorem ipsum dolor sit amet.
</div>
```
<Canvas withSource={ SourceState.NONE }>
  <div
    className="border-black font-sans-1"
    style={{ 'padding': '2rem', 'border': '1px solid' }}>
    Lorem ipsum dolor sit amet.
  </div>
  <div
    className="border-pink-300 font-sans-1"
    style={{ 'padding': '2rem', 'border': '1px solid' }}>
    Lorem ipsum dolor sit amet.
  </div>
</Canvas>
