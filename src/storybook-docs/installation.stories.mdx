# Selamat Datang di Jabar Digital Service Sistem Desain

**Jabar Digital Service Sistem Desain** adalah _open source_ JDS yang digunakan dalam mendesain dan mengembangkan aplikasi *web* maupun *mobile* dalam lingkungan pemerintahan Jawa Barat. Sistem desain ini terdiri dari komponen siap pakai dan kumpulan kode pendukungnya serta pedoman penggunaan untuk masing-masing komponen tersebut.

## Catatan
Sistem desain ini masih diperuntukan digunakan pada *framework* VueJS selain itu kami juga berharap agar bisa mendukung penggunaan di *framework* lain dengan membuat *library* ini menjadi *web component* yang sedang kami kembangkan sampai saat ini.

## Instalasi
Untuk melakukan instalasi sistem desain ini anda dapat melakukannya dengan menggunakan aplikasi *package manager* NPM, lakukan perintah di bawah ini dengan aplikasi **CLI** (**C**ommand **L**ine **I**nterface) pada *root* direktori proyek anda.

```powershell 
npm i @jabardigitalservice/jds-design-system
```
instalasi di atas menunjukan anda menginstal versi terbaru dari *library* sistem desain kami. Untuk menginstal versi yang spesifik, lakukan perintah seperti contoh
```powershell
npm i @jabardigitalservice/jds-design-system@0.0.3--beta.2
```
atau

```powershell
npm i @jabardigitalservice/jds-design-system@0.1.0
```
anda bisa melihat versi yang tersedia sistem desain kami di *package manager* [Versi](https://www.npmjs.com/package/@jabardigitalservice/jds-design-system?activeTab=versions).
Setelah anda berhasil melakukan instalasi, buka file main.js yang berarti *entrypoint* pada proyek anda, lalu tambahkan kode dibawah ini.
```javascript 
import { DesignSystem } from '@jabardigitalservice/jds-design-system'
Vue.use(DesignSystem)
```
contoh gambar pada *code editor* Visual Studio Code

### Desain Token
Apabila dan ingin menggunakan desain token kami, anda dapat menambahkan kode di bawah pada *entrypoint* main.js
```javascript
import '@jabardigitalservice/jds-design-system/css/jds-design-system.css'
```
### Ikon
Apabila anda ingin menggunakan ikon sistem kami, anda dapat meng-*import* ikon tersebut ke dalam kode anda seperti berikut ini.
untuk penggunaan API Ikon Sistem anda dapat melihatnya disini [Ikon Sistem](https://master--60d0320b29870700395d5ad8.chromatic.com/?path=/docs/components-icon--default)
```javascript
<template>
  <bell />
</template>

import bell from '@jabardigitalservice/jds-design-system/icons/bell'

export default {
  components: {
    bell
  }
}
```
### Contoh
Contoh *setup* pada *entrypoint* main.js VueJS
```javascript
import Vue from 'vue'
import App from './App.vue'

import '@jabardigitalservice/jds-design-system/css/jds-design-system.css'

/**
* register all components globally
*/

import { DesignSystem } from '@jabardigitalservice/jds-design-system'
Vue.use(DesignSystem)

/**
* register component ala carte
*/

// import { Button } from '@jabardigitalservice/jds-design-system/esm'
// Vue.component('JdsButton', Button)
// Vue.component('JdsInputText', InputText)

new Vue({
el: '#app',
render: (h) => h(App),
}).$mount('#app')
```
